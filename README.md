# 16-0 — All-Time IPL Draft Game

A complete Next.js single-page game inspired by the provided screenshots.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Next.js.

## Game loop

1. Choose mode, lineup, and rerolls.
2. Spin to reveal a franchise + season.
3. Draft one player into the required XI slot.
4. Complete 11 picks and chase a perfect 16-0 season.

No external API or images are required. Everything runs locally.
